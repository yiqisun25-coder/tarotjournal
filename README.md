# 我的塔罗记录 · Personal Tarot Journal

一个纯静态的个人塔罗日记网站：记录每次抽牌的问题、牌面、解释和之后的回看，配合一套 53 天的塔罗学习课程。手机和电脑通过 JsonBin 自动双向同步。

**在线地址**：https://yiqisun25-coder.github.io/tarotjournal/

## 功能

### 📓 抽牌记录
- 记录日期、主题、情绪、问题、隐藏担忧、牌阵、整体解释、提醒与行动
- 每张牌可填位置、牌名（78 张标准牌自动补全）、正逆位、第一眼感受、个人解释、关键词
- 牌面以花色配色的小牌面呈现：权杖🔥 / 圣杯💧 / 宝剑⚔️ / 星币🪙 / 大阿卡纳✨，逆位符号倒转
- 可附一张牌桌照片（自动压缩到约 30KB 以便同步）
- **回看**：过一段时间回来补充"后来发生了什么、当时哪里准、哪里偏"
- 搜索、按主题/回看状态筛选、统计最常出现的牌

### 📅 每日一牌 · 塔罗学习（53 天一轮）
按《塔罗学习手册》的节奏自动排课，用日期计算当天单元，所有设备显示同一课：

| 天数 | 内容 |
|---|---|
| 1–22 | 大阿卡纳，每天一张：视觉锚点 / 正逆位 / 对应 / 核心命题 / 生活语境 / 阴影面 / 组合练习 |
| 23–26 | 四花色骨架（我想做 / 我感觉 / 我认为 / 我拥有） |
| 27–36 | 十个数字，每天四张同数字牌图，先推后对照 |
| 37–39 | 九张例外牌（图像压过逻辑，必须记） |
| 40–43 | 宫廷牌：4 位阶 × 4 花色 + 人/态度/事三读法 |
| 44–53 | 牌阵实战：三张牌阵、位置改写牌义、凯尔特十字、真人实测 |

- 显示 1909 年韦特原版牌图（公有领域），用于认牌
- 单张牌的日子会显示"我抽到过 N 次"，一键搜出自己的相关记录
- 每天一条强制动作（大牌日：把今天的牌和昨天连成 30 字的故事）

### ☁️ 云端同步（JsonBin）
- 保存、删除、回看后自动上传；打开页面、切回页面、重新联网时自动下载合并
- **先拉取、合并、再上传**：多设备不会互相覆盖
- 删除用同步标记（tombstone）传播，不会被其他设备"复活"
- JsonBin 免费版整库上限 100KB：容量不够时自动从最旧记录开始去掉照片（照片保留在本机），文字记录永远优先同步
- **配对链接**：点右上角 ☁️ 角标 →「复制手机配对链接」，新设备打开一次即自动连接，无需手输密钥
- API Key 只存在各设备浏览器的 localStorage 里，不出现在代码中
- 支持 JSON 导出 / 导入备份

## 文件结构

```
index.html        页面结构
style.css         样式
app.js            记录、同步、渲染逻辑
learning-data.js  53 天课程数据（源自 塔罗学习手册.md）
tarot-data.js     78 张牌义词条（当前未加载，保留备用）
cards/            78 张韦特牌图（360px JPEG）
塔罗学习手册.md    学习手册原文
```

纯 HTML/CSS/JS，无构建、无依赖，任何静态托管（GitHub Pages / Vercel）都能跑。

## 同步设置（仅首次）

1. 注册 [jsonbin.io](https://jsonbin.io)（免费），复制 API Key（X-Master-Key）
2. 打开网站 → 点右上角 ☁️ → 填 API Key → 「创建云端库」
3. 其他设备：在已连接的设备上点「复制手机配对链接」，用新设备打开一次即可
